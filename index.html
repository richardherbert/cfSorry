cfSorry

I am a ColdFusion component (cfc) that provides a convenient wrapper around the Sorry™ API.

I work with Adobe ColdFusion 10+, 11+, 2016+, Lucee 4.5+ and Railo 4.2+

Getting Started

  1. First you'll need an account with Sorry™. It's free to sign up and you'll get an API token which you'll need later.
  2. Visit the cfSorry page on GitHub, download the latest version and then drop the component cfSorry.cfc into the webroot of your website.
  3. Instantiate the cfSorry.cfc component.
    This can be done with:
    • Your dependency injection frameworks of choice (WireBox or DI/1)
    • Assigning it to an application scoped variable in your Application.cfc event onApplicationStart() with… 
      public boolean function onApplicationStart() {
	application.cfSorry = new cfSorry("myAPIToken");

	return true;
}
    • Or on demand where you need the service with… 
      cfSorry = new cfSorry("myAPIToken")
  4. Use the component where you need in your code like…
    <cfset pages = cfSorry.getPages()>
    or 
    <cfscript>
	pages = cfSorry.getPages();
</cfscript>
